## About The Project

LP2 is a web-based gaming platform that brings together a variety of browser games in one convenient location. Built with modern web technologies, it provides a smooth and enjoyable gaming experience directly in your browser.

## Features

- Diverse collection of browser games
- Responsive design for all devices
- Fast loading and smooth performance
- Clean and intuitive user interface
- Free to use and open source

### Run Locally

1. Clone the repository
```bash
git clone https://github.com/Olibot1107/games.git
```

2. Navigate to the project directory
```bash
cd games
```

3. run
```bash
npm i && node main.js
```
